Dashboard Documentation

This is the documentation for the dashboard of the meeting management system. We are using C4 model to document the system. The C4 model is a way to document software architecture based on 4 levels of abstraction. The levels are context, container, component and code.

The dashboard is the main page of the system. It is the first page that the user sees when they log in. It contains a summary of the user's meetings and the meetings that they are invited to.
